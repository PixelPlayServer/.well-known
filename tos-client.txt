Términos de Servicio - PixelPlay Launcher
Última actualización: [Fecha]

1. Aceptación de los Términos
Al descargar, instalar o utilizar PixelPlay Launcher ("el Launcher"), usted acepta estar vinculado por estos Términos de Servicio. Si no está de acuerdo con estos términos, no utilice el Launcher.

2. Descripción del Servicio
PixelPlay Launcher es una aplicación que permite gestionar y ejecutar versiones modificadas de Minecraft con mods y resource packs. El servicio incluye funcionalidades de gestión de archivos, configuración de juegos y telemetría del sistema.

3. Recopilación y Uso de Datos
3.1 Información Recopilada
Al utilizar PixelPlay Launcher, recopilamos automáticamente la siguiente información:

Información del Sistema:

Especificaciones del hardware (CPU, memoria total y disponible)
Sistema operativo y arquitectura
Identificadores únicos del dispositivo (HWID, Machine ID)
Información de red (dirección IP externa, direcciones MAC, interfaces de red)
Nombre del equipo (hostname)
Tiempo de actividad del sistema
Información de la Aplicación:

Versión del Launcher utilizada
Mods instalados (nombres, versiones y tamaños)
Resource packs instalados
Archivos .jar en el directorio de versiones
Método de autenticación utilizado
Información de Usuario:

Nombre de usuario del sistema operativo
Método de autenticación (Microsoft, etc.)
Dirección IP local
3.2 Propósito de la Recopilación
Los datos recopilados se utilizan exclusivamente para:

Mejorar la funcionalidad y rendimiento del Launcher
Proporcionar soporte técnico
Detectar y prevenir uso indebido o actividades maliciosas
Mantener la seguridad y estabilidad del servicio
Generar estadísticas agregadas y anónimas
3.3 Transmisión de Datos
Los datos recopilados se transmiten a una red privada de administradores selectos. Esta información se maneja de manera confidencial y no se utiliza para fines maliciosos o comerciales no autorizados.

4. Uso Aceptable
4.1 Conducta Permitida
Usted puede utilizar el Launcher para:

Ejecutar versiones modificadas de Minecraft con mods compatibles
Gestionar sus configuraciones de juego
Personalizar su experiencia de juego mediante resource packs
4.2 Conducta Prohibida
Está estrictamente prohibido:

Modificar, descompilar o realizar ingeniería inversa del Launcher
Utilizar el Launcher para actividades no autorizadas por los términos de servicio
Interferir con el funcionamiento normal del servicio
Intentar acceder a datos de otros usuarios
Usar el Launcher para distribuir malware o contenido malicioso
Violar los términos de servicio de Minecraft o Microsoft
Realizar actividades que puedan dañar la reputación o funcionamiento del servicio
5. Sanciones y Medidas Disciplinarias
5.1 Derecho de Sanción
Nos reservamos el derecho de imponer sanciones apropiadas a usuarios que violen estos términos, incluyendo pero no limitándose a:

Advertencias formales
Suspensión temporal del servicio
Prohibición permanente del uso del Launcher
Bloqueo de acceso basado en HWID o dirección IP
5.2 Proceso de Sanción
Las sanciones se implementan de forma justa y proporcional a la gravedad de la infracción. Los usuarios pueden ser sancionados basándose en:

Análisis de los datos de telemetría recopilados
Reportes de la comunidad
Detección automatizada de actividades sospechosas
Violaciones repetidas de los términos
6. Privacidad y Seguridad
6.1 Protección de Datos
Implementamos medidas de seguridad técnicas y organizacionales para proteger sus datos personales contra acceso no autorizado, alteración, divulgación o destrucción.

6.2 Retención de Datos
Los datos recopilados se mantienen durante el tiempo necesario para cumplir con los propósitos descritos en estos términos.

6.3 Acceso Limitado
Solo el personal autorizado y los administradores selectos tienen acceso a los datos recopilados, y están obligados a mantener la confidencialidad.
